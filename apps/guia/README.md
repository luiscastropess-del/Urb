# Prospector de Locais & Guia Holambra

Sistema híbrido de prospecção de dados geográficos e marketplace para guias turísticos.

## Documentação de Desenvolvedor

Para estender as funcionalidades do sistema, consulte os seguintes guias:

*   [Guia de Desenvolvimento de Plugins](./GUIDE_PLUGIN_DEVELOPMENT.md): Instruções passo a passo para criar novos módulos, widgets e integrações.
*   [Arquitetura do Sistema](./AI_ARCHITECTURE.md): Detalhes técnicos sobre a stack, banco de dados e diretrizes de código.

## Tech Stack
*   **Next.js 15+** (App Router)
*   **Prisma ORM** (PostgreSQL)
*   **Tailwind CSS v4**
*   **Lucide React** (Icons)
*   **Overpass API** (OpenStreetMap)
